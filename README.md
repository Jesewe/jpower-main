### jpower
**Это модуль для Python, который предоставляет список ошибок.**

**Чтобы их написать в коде, импортируйте модуль:**
```python
import jpower

jpower.error1()
jpower.error2()
jpower.error3()
jpower.error4()
jpower.error5()
jpower.error6()
jpower.error7()
jpower.error8()
jpower.error9()
jpower.error10()
jpower.error11()
```
